To play the game you have to have the following installed:
- SDL2
- SDL2_image
- SDL2_Mixer

Installing SDL2 and SDL2_image
Install it using the terminal commands on linux:

username:~$ sudo apt-get install libsdl2-dev
username:~$ sudo apt-get install libsdl2-image-dev
username:~$ sudo apt-get install libsdl2-mixer-dev

Play the game
in terminal:
step to the right directory
create the game using:
username:~/projektet$ make game

play the game using:
username:~/projektet$ ./game.out
